# Competitor Analysis: Pokémon TCG Proxy Tools

> Last updated: 2026-04-12

---

## Direct Competitors (Web-Based Proxy Generators)

### 1. **Limitless TCG Proxy Printer**
- **URL:** https://limitlesstcg.com/tools/proxies
- **Description:** The most well-known and widely used proxy printer in the competitive Pokémon TCG community
- **Features:**
  - Search individual cards or paste deck lists
  - Print proxies for Standard and Expanded formats
  - Simple, clean interface
  - Part of the larger Limitless TCG ecosystem (tournament platform, deck builder)
- **Strengths:**
  - Trusted brand in competitive community
  - Integration with Limitless tournament results
  - "The best Pokemon TCG proxy printer in the web" - quoted from Hamatti's Toolkit
- **Weaknesses:**
  - Limited customization options (no variant selection, no size options)
  - Basic print layout only
  - No deck saving/sharing features
- **User Base:** Competitive players, tournament goers
- **Our Advantage:** Better UX, variant selection, multiple sizes, deck builder with persistence

---

### 2. **Proxycroak**
- **URL:** https://proxycroak.com/
- **Description:** Simple tool to print proxies straight from a decklist
- **Features:**
  - Paste deck list, get printable proxies
  - Focus on simplicity and speed
- **Strengths:**
  - Very simple, no registration required
  - Fast workflow for existing deck lists
- **Weaknesses:**
  - No search functionality
  - Must have deck list already
  - No visual preview of cards
- **User Base:** Players with existing deck lists
- **Our Advantage:** Full search, visual card browser, deck building UI

---

### 3. **Proxxied (proxxied.com)**
- **URL:** https://proxxied.com/
- **Description:** Multi-TCG proxy maker (MTG, Pokémon, Yu-Gi-Oh)
- **Features:**
  - Search by name
  - Browse art from every set/expansion
  - High-resolution print-ready PDFs
  - Custom card maker with dimensions
  - Custom bleed and layout options
- **Strengths:**
  - Professional quality output
  - Multiple TCGs (cross-platform network effects)
  - No account required
  - Custom card dimensions
- **Weaknesses:**
  - May be overwhelming for casual users
  - Generic across TCGs (not Pokémon-specific)
- **User Base:** Multi-TCG players, professional proxy users
- **Our Advantage:** Pokémon-specific features, set filtering, variant selection

---

### 4. **TCG Stacked / ProxyCreatures**
- **URL:** https://www.tcgstacked.com/
- **Description:** Multi-TCG proxy platform with premium features
- **Features:**
  - MTG, Pokémon, Lorcana, One Piece, Star Wars, Digimon
  - Mobile app (iOS/Android)
  - Pro tier: $3.33/month ($40/year)
  - Card price history charts
  - AI generation credits (for MTG)
  - Unlimited deck creation
- **Strengths:**
  - Mobile app presence
  - Multi-TCG support
  - Premium features for power users
- **Weaknesses:**
  - Paid subscription for full features
  - Less focused on Pokémon specifically
- **User Base:** Multi-TCG collectors, mobile-first users
- **Our Advantage:** Free tier is more generous, Pokémon-focused

---

### 5. **PokéPlaceholder (shermaza.github.io)**
- **URL:** https://shermaza.github.io/pokeplaceholder.com/
- **Description:** Binder placeholder generator (different use case)
- **Features:**
  - Create placeholder cards for binders
  - Track missing cards in collection
  - Printable PDF (8.5" × 11")
  - Filter by Pokédex number, generation
- **Strengths:**
  - Unique use case (collection management)
  - Image or text-only labels
  - Customizable cards per page
- **Weaknesses:**
  - Not for playtesting (placeholders only)
  - No deck building features
- **User Base:** Collectors organizing binders
- **Our Advantage:** Playtesting focus, full card images, deck builder

## Physical/Shipping Competitors (Not Direct, But Relevant)

### 6. **ProxyKing (proxyking.biz)**
- **URL:** https://proxyking.biz/pokemon-proxies/
- **Description:** Physical proxy cards shipped to you
- **Price:** ~$0.80 per card, bundles available (45x full holo vintage for ~$36)
- **Quality:** High-quality card stock, professional printing
- **Turnaround:** Ships same day, arrives in days
- **Use Case:** Players who want physical cards without printing themselves
- **Our Advantage:** Free, instant, no shipping wait, DIY

### 7. **OP Proxies TCG**
- **URL:** https://opproxiestcg.com/
- **Description:** Graded proxy cards with custom PSA-style labels
- **Price:** Premium (~¥1,042 per card = ~$7 USD)
- **Features:** Custom graded slabs, GEM MT 10 labels
- **Use Case:** Collectors wanting display pieces
- **Our Advantage:** Free, functional for playtesting

### 8. **PrintingProxies.com**
- **URL:** https://www.printingproxies.com/
- **Description:** Professional proxy printing service
- **Features:** Custom sizes, various finishes
- **Use Case:** Bulk printing for tournaments
- **Our Advantage:** Free DIY option, immediate gratification

## Alternative/Adjacent Tools

### 9. **Hamatti's Proxy Printer**
- **URL:** https://hamatti.org/proxy
- **Description:** Simple drag-and-drop image printer
- **Features:**
  - Drag any card image
  - Prints 9 cards per A4 page
  - Purely local (no data sent)
  - Designed for sleeve compatibility
- **Strengths:**
  - Privacy-focused (no server)
  - Simple, works with any image
  - Fast
- **Weaknesses:**
  - No search/database
  - Must find images yourself
- **Our Advantage:** Integrated database, search, deck management

### 10. **TCG Proxy Creator (tcg-proxy-creator.vercel.app)**
- **URL:** https://tcg-proxy-creator.vercel.app/
- **Description:** Create high-quality print data for TCG proxies
- **Features:**
  - Up to 600 DPI
  - Save cards locally
  - High resolution
- **Weaknesses:**
  - Basic functionality
  - No deck lists from database
- **Our Advantage:** Integrated card database, deck builder

## Threats to Watch

1. **Limitless TCG** could enhance their proxy tool with variants/sizes
2. **TCG Stacked** mobile app could add better Pokémon features
3. **The Pokémon Company** could crack down on proxy tools (legal risk)
4. **New entrant** with VC funding could out-feature you quickly
